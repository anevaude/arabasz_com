ESPN Event Wrap-ups

  • HTML
  • CSS
  • jQuery
  • Expression Engine
  • Mustache
  • Foundation

A collection of sites summarizing the X Games and ESPYs Awards. The primary audience is event sponsors, who use the website to see how their brand was promoted.

Each site contains a large collection of media files from the event, including photos, videos and PDFs. Sponsors can log in to a password-protected area to view marketing statistic documents.

The original version of the site was built in 2012 on the Expression Engine CMS. It was revamped in 2015 using Mustache and Foundation, resulting in a much simpler and easier-to-maintain experience.
