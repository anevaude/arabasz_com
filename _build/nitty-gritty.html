Nitty Gritty Details

Wondering how I code? Here's what I used to build this site.*


Technologies

  • HTML5 and CSS3 with SCSS. Mobile-first and responsive.
  • Handlebars JS with Mustache for dynamic elements and templating.
  • PostCSS Autoprefixer to parse my CSS and add vendor prefixes.
  • SUSY for pre-structuring responsive grids.
  • Grunt as my Javascript task runner to compile everything.
  • Sublime Text 2 for text editing.

* Please note that at the time, all these technologies were "modern". So if you're reading this in the year 2050, you may be having a good laugh!
