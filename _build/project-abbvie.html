Overview

AbbVie is one of America's largest pharmaceutical companies. This project created a responsive design system for their Pharmacy Solutions Manager, a tool that would need to be accessible on mobile devices.

The design system included Abbvie's branding guidelines, documentation, and code snippets for typography, color, and logo usage.

Team & Role

I served as a Creative Technologist, working closely with the design team to create an on-brand collection of elements for AbbVie.
